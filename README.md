# 🛡️ InventarioSeguro

Aplicación web desarrollada en **ASP.NET Core MVC** con **SQLite** y **Entity Framework Core** para la gestión de inventario.  
Incluye un **CRUD completo de productos** y un **Dashboard visual con gráficos (Chart.js)** para estadísticas.

---

## 🚀 Características
- CRUD de productos (Crear, Leer, Editar, Eliminar).
- Validaciones en formularios (nombre obligatorio, cantidad > 0, precio > 0).
- Interfaz moderna con **Bootstrap**.
- Dashboard con:
  - 📊 Gráfico de barras: cantidad por producto.
  - 🥧 Gráfico de pastel: distribución proporcional del inventario.

---

## 🛠️ Tecnologías utilizadas
- **ASP.NET Core MVC 8**
- **Entity Framework Core**
- **SQLite**
- **Bootstrap 5**
- **Chart.js**
